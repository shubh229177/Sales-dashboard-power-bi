# 📊 Sales Analysis Dashboard (Power BI)

## 📌 Project Overview

This project focuses on analyzing sales data and building an interactive dashboard using Power BI. The dataset was cleaned and transformed before creating meaningful visualizations to extract business insights.

---

## 🧹 Data Cleaning

* Removed duplicates and handled missing values
* Standardized column formats (dates, categories, etc.)
* Created additional columns like **Year** and **Month** for better analysis

---

## 📂 Dataset Columns

* Order_ID
* Order_Date
* Region
* Category
* Product
* Sales
* Profit
* Quantity
* Ship_Days
* Year
* Month

---

## 📊 Dashboard Features

* 📈 Total Sales, Profit & Quantity Overview
* 🌍 Region-wise Sales Analysis
* 📦 Category & Product Performance
* 📅 Monthly & Yearly Trends
* 🚚 Shipping Days Analysis
* 🔎 Interactive Filters & Slicers

---

## 💡 Key Insights

* Identified top-performing regions and products
* Observed seasonal trends in sales
* Analyzed profit distribution across categories
* Found patterns in shipping time vs performance

---

## 🛠 Tools & Technologies

* Power BI
* Excel / CSV
* Data Cleaning

---

## 🚀 Conclusion

This dashboard helps in understanding business performance and supports data-driven decision-making through interactive visuals.
